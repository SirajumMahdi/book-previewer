A WordPress plugin that adds a book preview feature to your WooCommerce products.

== Description ==

The Book Previewer allows bookstore websites to display a short preview of each book. It enhances customer engagement and encourages them to explore the book visually.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/book-previewer` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =

Yes, this plugin requires WooCommerce to function properly.
